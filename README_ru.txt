Как вы уже знаете, daemon_n выпустил прекрасный плагин на кастомизацию объектов - RMG_CustomizeObjectProperties, добавив также и новые типы объектов в базовый WoG-мод, такие, как склады/беседка/магический колизей. Я решил расширить этот список следующими объектами:


- форт на холме (тип 1 - не позволяет улучшать существ выше 5 уровня)
- храм лояльности
- преобразователь скелетов
- водопой
- минеральный источник
- лачуга отшельника
- старьёвщик
- лаборатория чернокнижника
- старатель
- проводник
- древняя лампа
- обсерватория (в т.ч. наземная)
- могила


Добавлен новый тип объект, не встречающийся в других модах и дополнениях HoMM3, но существующий в HoMM4:

- толкователь снов


Добавлена новая графика для уже существующих объектов:

- AVSutop5.def - Утопия Драконов для болотистой почвы
- AVSutop8.def - Утопия Драконов для лавовой почвы
- AVXfyth7.def - Фонтан Молодости для подземной почвы
- AVTmystd.def - Мистический Сад для грязевой почвы
- AVXstbl7.def - Конюшни для лавовой и подземной почвы
- AVXtrek2.def - Древо Мудрости для песчаной почвы
- AVXtrek5.def - Древо Мудрости для болотистой почвы
- AVXtrek3.def - Древо Мудрости для каменистой почвы (взято из базовой сборки WoG)
- AVXtrek7.def - Древо Мудрости для лавовой и подземной почвы
- HAExit01.def - Врата Подземного Мира для подземной почвы
- avmoilpm.def - Водяное Колесо для подземной почвы


Для всех объектов, кроме преобразователя скелетов, старьёвщика и лаборатории чернокнижника написана логика для ИИ.
Генерацию любого объекта можно кастомизировать через RMG-диалог в игре или через json-файл в папке /Lang.
